## 📂 ARQUIVOS DO PROJETO

### Frontend (HTML)
```
index.html              Tela de login
dashboard.html          Dashboard principal
cadastro.html           Criar/editar indicações
indicacoes.html         Listar indicações
pagamentos.html         Registrar pagamentos
relatorios.html         Gerar relatórios
```

### Estilos
```
main.css               Todos os estilos do sistema
```

### Backend (JavaScript)
```
firebase-config.js     Configuração do Firebase
auth.js                Autenticação de usuários
indicacoes.js          Lógica de indicações
```

### Configuração
```
firestore.rules        Regras de segurança (copiar para Firebase)
package.json           Informações do projeto
vercel.json            Configuração do Vercel
.gitignore             Arquivos que não vão ao GitHub
```
