## 🚀 3 PASSOS PARA COMEÇAR

### PASSO 1: Baixar Arquivos
1. Crie pasta `indicacoes` no seu computador
2. Coloque todos esses arquivos lá

### PASSO 2: Enviar para GitHub
```bash
cd indicacoes
git add .
git commit -m "Setup inicial"
git push origin main
```

### PASSO 3: Configurar Vercel
1. Acesse https://vercel.com
2. "Add Project" → Selecione "indicacoes"
3. Clique em "Deploy"
4. **PRONTO! Seu site está online!** 🎉

---

## ⚙️ UMA VEZ: Configurar Firebase

### Firebase Rules
1. Firebase Console → Firestore Database → Regras
2. Copiar `firestore.rules`
3. Publicar

### Criar Usuário Admin
1. Firebase Auth → Add User → email/senha
2. Copiar UID gerado
3. Firestore → New Document em `usuarios`
4. ID = UID copiado
5. Adicionar campos: nome, email, perfil, status, senhaTemporaria, criadoEm
