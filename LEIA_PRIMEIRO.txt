                            COMECE AQUI - 3 PASSOS
================================================================================

PASSO 1️⃣ - ENVIAR PARA GITHUB
─────────────────────────────────────────────────────────────────────────────
Abra Terminal/Cmd e execute:

    git add .
    git commit -m "Setup inicial"
    git push origin main

Pronto! Seus arquivos estão no GitHub.


PASSO 2️⃣ - FAZER DEPLOY NO VERCEL
─────────────────────────────────────────────────────────────────────────────
1. Acesse: https://vercel.com
2. Faça login (ou crie conta)
3. Clique em "Add New Project"
4. Selecione seu repositório "indicacoes"
5. Clique em "Deploy"

Aguarde 2-3 minutos... Seu site estará ONLINE! 🌐


PASSO 3️⃣ - CONFIGURAR FIREBASE
─────────────────────────────────────────────────────────────────────────────
Acesse: https://console.firebase.google.com

3.1 - Publicar Firestore Rules
    → Firestore Database
    → Regras
    → Copiar conteúdo de "firestore.rules"
    → Publicar

3.2 - Criar Usuário
    → Authentication
    → Add User
    → Email: admin@empresa.com
    → Senha: SenhaForte123!
    → COPIAR O UID (importante!)

3.3 - Registrar em Firestore
    → Firestore Database
    → Criar collection "usuarios"
    → New Document
    → Document ID = UID copiado acima
    → Adicionar campos:
        • nome: "Admin"
        • email: "admin@empresa.com"
        • perfil: "admin"
        • status: "ativo"
        • senhaTemporaria: false
        • criadoEm: (data/hora agora)
    → Save
